# Live iteration checklist

Copy mentally each session:

- [ ] Features: `remote,capture`  
- [ ] Port: 15702 (or registered target)  
- [ ] BRP discover succeeds  
- [ ] At least one query returns entities  
- [ ] Capture saved and **visually inspected**  
- [ ] Defect list written (code vs art vs lighting)  
- [ ] One fix applied  
- [ ] Recapture confirms fix  

## Acceptance examples

**2D movement:** player sprite leaves start position when keys pressed (or after BRP mutate of translation).  
**3D movement:** capsule moves on ground; still lit and in frame.  
**Menu:** capture shows menu; after transition, gameplay scene.  
**Art drop:** new texture visible on player, not pink/missing.
